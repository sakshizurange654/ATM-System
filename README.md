# ATM System Project

This repository contains a **Python-based ATM System project** that simulates basic ATM functionalities.  
The project is created for learning and academic purposes to understand real-world banking operations using Python.

##  Project Features
- User authentication using PIN
- Balance inquiry
- Cash withdrawal
- Cash deposit
- Exit option

##  Objective
- To understand how an ATM system works
- To practice Python programming concepts
- To improve logical thinking and problem-solving skills
- To apply conditional statements and loops in a real-world scenario

## Technologies Used
- Python 3

##  Concepts Used
- Variables
- Conditional statements (`if-else`)
- Loops
- Functions
- User input/output

## How to Run the Project
1. Make sure Python is installed on your system
2. Clone this repository or download the ZIP
3. Run the Python file using:
   ```bash
   python atm_system.py
